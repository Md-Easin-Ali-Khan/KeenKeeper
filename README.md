# 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Description

KeenKeeper is a modern web application designed to help users maintain meaningful relationships with their friends. It allows users to track interactions, set communication goals, and visualize their social activity through an intuitive and responsive interface.

---

## 🛠️ Technologies Used

- **Next.js (App Router)** – for building the full-stack React application
- **React** – component-based UI development
- **Tailwind CSS** – for fast and responsive styling
- **Lucide React** – for clean and modern icons
- **Recharts** – for data visualization (Pie Chart)
- **React Hot Toast** – for interactive notifications
- **Context API** – for global state management (timeline system)

---

## ✨ Key Features

### 1. 📊 Friendship Tracking System

- Add and monitor interactions like Call, Text, and Video
- Automatically logs activities into a timeline
- Displays interaction history with date and type

### 2. 🧠 Smart Friend Management

- View detailed friend profiles with status, tags, and goals
- Track “days since last contact” and next due date
- Visual status indicators (Overdue, Almost Due, On-Track)

### 3. 📈 Analytics Dashboard

- Interactive Pie Chart showing communication patterns
- Real-time data updates based on user interactions
- Clean and responsive visualization using Recharts
